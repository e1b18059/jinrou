ルール説明

  • このゲームは人間側と人狼側に分かれ、5分間の議論を行う昼時間と能力を行使する夜時間を繰り返してお互いの陣営の勝利を目指すものになっています。
  • 平民…何の能力も持たない役職です。議論で勝利に貢献しましょう。
  • 占い師…夜時間毎に任意のプレイヤーが人狼か人間かを知ることができる役職です。ただし後述する狂人の判別はできません。
  • 霊媒師…夜時間毎に死んだプレイヤーが人狼か人間かを知ることが出来る役職です。占い師同様狂人の判別はできません。
  • 騎士…夜時間毎に任意のプレイヤーを狼の襲撃から守ることができます。
  • 人狼…夜時間毎に任意のプレイヤーを襲撃し、騎士に守られていなければ殺害できます。人狼が二人いる場合でも襲撃できるのは夜時間毎に一人だけです。
  • 狂人…勝利条件が人狼側の人間です。勝利条件以外は平民と同様特に能力は無く、誰が人狼かを知ることは出来ません。議論の妨害や誘導が主な目的になります。

ルール詳細

-------------------------------★勝利条件★-----------------------------------

  • ●人間側…人狼を全て殺せば勝利です。六人以上で遊ぶとき人狼が二人になります。
  • ●人狼側…人間と人狼が同数になれば勝利です。人間の数には狂人も含まれます。
